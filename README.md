# Serva Claude Marketplace

Claude Code plugins for the Serva homelab ecosystem.

## Plugins

| Plugin | Description |
|---|---|
| [mac-setup](plugins/mac-setup/) | Interactive Mac development environment setup |

## Installing a Plugin

```bash
cc --plugin-dir /path/to/plugins/mac-setup
```

Or copy the plugin directory to your project's `.claude/plugins/` folder.
